# 渝畅行预约页面-仅供学习



链接支持参数

| 参数名         | 默认值   | 描述         |
| -------------- | -------- | ------------ |
| `station_name` | `光电园` | 站点名称     |
| `line_name`    | `6号线`  | 线路名称     |
| `start_time`   | `18:20`  | 预约开始时间 |
| `end_time`     | `18:30`  | 预约结束时间 |

举例：
`https://hyc8801.github.io/appointment/#/appointmentRecordDetail?station_name=光电园&line_name=6号线&start_time=17:30&end_time=17:40`
